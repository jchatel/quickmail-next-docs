# Importing from a LinkedIn Post 📲

**Tip:** You can also import leads from a different person's LinkedIn post

Users can now import leads from LinkedIn posts. This makes it easier to capture engaged prospects directly and streamline your outreach efforts.

## How to Import leads from a LinkedIn post?

**Step 1.** Add a LinkedIn account to QuickMail. Go to Channels → LinkedIn → + LinkedIn. This guide about Adding LinkedIn accounts might come in handy.

**Step 2.** On a different browser tab, go to the LinkedIn post from which you’d like to import leads and copy the link to post.

![screenshot](../images/000_file-mr2uTnY1zc.png)

**Step 3.** Go back to QuickMail → List → + Lead → LinkedIn post

![screenshot](../images/001_file-QSgxWzJRxo.png)

**Step 4.** Paste the LinkedIn post link into QuickMail.

![screenshot](../images/002_file-u04xOzQS8u.png)

**Step 4.** Map the leads' properties and add custom properties or tags if you'd like.

![screenshot](../images/003_file-WodoXIZ5X3.png)

**Step 5.** Select the campaign where you would like to add the leads (optional) and click 'Import'

![screenshot](../images/004_file-99FuoqOvCX.png)

**Step 6.** An import report will be sent shortly via email where you can see how many leads have been imported or rejected.

![screenshot](../images/005_file-ztNp74NRPC.png)
