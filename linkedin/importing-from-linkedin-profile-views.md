# Importing from LinkedIn Profile Views 👀

You can automatically import leads who viewed your LinkedIn profile into QuickMail. This helps you quickly follow up with interested leads and increase your chances of conversion.

**In this article:**

- How to enable LinkedIn profile view imports?

## How to Enable LinkedIn Profile View Imports?

Go to **LinkedIn** → click on the LinkedIn account → **Receiving** tab → check **Create leads from profile viewers** → optionally select a campaign to add the leads to.

Alternatively, you can use the browser extension to set this up:
https://help.quickmail.com/linkedin/quickmails-linkedin-browser-extension/
