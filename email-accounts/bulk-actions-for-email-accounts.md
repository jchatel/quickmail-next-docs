# Managing Email Accounts with Bulk Actions

**In this article:**

- Why use bulk actions?

- Which bulk actions are available?

- How to take bulk actions?


## Why Use Bulk Actions?

Bulk actions allow you to manage multiple email accounts and LinkedIn accounts at the same time, saving time by reducing the need to update each account individually.


## Which Bulk Actions Are Available?

**Email**

- Pause sending

- Resume sending

- Set daily email limit

- Set minimum delay between emails

- Set random delay between emails

- Set sender name 

- Set email signature

- Set custom domain tracking

- Delete email account

## How to Use Bulk Actions?

Go to **Email** → select multiple email accounts → select the bulk action to take.
