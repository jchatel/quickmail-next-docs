# Setting a Daily Sending Limit

**In this article:**

- Why limit daily email sends?

- How does the daily sending limit work?

- How to set it up for a single inbox?

- How to set it up for multiple inboxes?

## Why Limit Daily Email Sends?

The daily sending limit allows you to control how many emails are sent from an inbox each day. This is useful for several reasons:

- **Warming up a new inbox** — new inboxes should be warmed up gradually to build a good sender reputation. Being able to adjust the limit makes it easy to increase sending volume incrementally as the inbox establishes its reputation.

- **Spreading campaign volume across multiple inboxes** — different inboxes may have different sending capacities. Setting individual limits makes it easy to control how much of a campaign's volume is handled by each inbox.

- **Reserving capacity for emails outside QuickMail** — email service providers impose daily sending limits to prevent abuse. If QuickMail uses up your entire daily limit, you will not be able to send manual emails or follow-ups outside of QuickMail.

**Note:** If you need to control the times and days a campaign can send emails, use send times. If you need to control the number of leads starting the campaign daily, use triggers. Here is a **video guide** on that.

## How Does the Daily Sending Limit Work?

The daily sending limit sets the maximum number of emails that can be sent from an inbox within a single day. The limit resets every midnight in the account's timezone.

To check your account timezone, go to **General Settings**.

## How to Set It Up for a Single Inbox?

Go to **Channels** → **Email** → click the inbox you want to configure → **Sending** → scroll down to **Maximum number of emails that QuickMail should send per day from this inbox** → set your preferred limit.

![screenshot](../images/000_file-AmFcYR8M9i.png)

## How to Set It Up for Multiple Inboxes?

Go to **Channels** → **Email** → select the inboxes you want to configure → **Set Daily Sending Limit**.

![screenshot](../images/001_file-fkXeiGzzTd.png)

Enter the limit → click **Confirm**.

![screenshot](../images/002_file-Ivw5pH6iL7.png)
