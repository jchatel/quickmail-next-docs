# Matching Email Service Provider

**In this article:**

- Why use this feature?

- How does it work?

- How to enable this feature?

## Why Use This Feature?

Matching an Email Service Provider (ESP) allows QuickMail to automatically send emails from inboxes that use the same provider as the recipient. This can improve deliverability by reducing the risk of emails being marked as spam and helping maintain a strong sender reputation.

## How Does It Work?

When leads start a campaign, they are automatically assigned to an inbox. This ensures all follow-up emails are sent from the same email address for consistency.

When the Match Email Service Provider feature is enabled, leads are only assigned to inboxes that use the same email provider as the lead. This means that if leads and inboxes belong to different providers, the sending distribution may be uneven — some inboxes may send more emails than others.

**Important:** This feature only applies to new leads added to the campaign. Existing leads will not be affected.

## How to Enable This Feature?

Go to the campaign → click the menu icon (three vertical dots) → **Settings**.

![screenshot](../images/000_file-e5ZgEwhNK6.png)

Go to the **Advanced** tab → enable **Match Prospect Email Provider**.
